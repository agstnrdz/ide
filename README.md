### geojson to js
correr script
```python geojson_to_js.py -i docs/layers_censo_gpkg_geojson/nombre_de_capa.geojson -o layers_js```
elegir carpeta de salida: ```-o```
correr sobre todas las capas de la carpeta: ```-i layers_censo_gpkg_geojson```
